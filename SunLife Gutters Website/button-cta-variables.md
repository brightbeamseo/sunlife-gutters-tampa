# SGT — Button & CTA CSS Variables

| Field | CSS Variable | Role |
|-------|-------------|------|
| `accent` | `--color-accent` | Filled primary buttons (call / phone) |
| `accentHover` | `--color-accent-hover` | Hover for filled primary buttons |
| `primaryCtaText` | `--color-primary-cta-text` | Text color for all phone / Call style filled buttons |
| `secondaryCtaBg` | `--color-secondary-cta-bg` | Outline "Request estimate" default background (often `transparent`) |
| `secondaryCtaBorder` | `--color-secondary-cta-border` | Outline border |
| `secondaryCtaText` | `--color-secondary-cta-text` | Outline default text |
| `secondaryCtaHoverBg` | `--color-secondary-cta-hover-bg` | Outline hover fill |
| `secondaryCtaHoverText` | `--color-secondary-cta-hover-text` | Outline hover text |
| `formButtonBg` | `--color-form-button-bg` | Hero + footer submit buttons |
| `formButtonHover` | `--color-form-button-hover` | Submit hover |
| `formButtonText` | `--color-form-button-text` | Submit label color |
